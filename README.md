Демонстративный Django-проект сайта для музыкальной группы с использованием
bootstrap.

Запуск сервера через python manage.py runserver в 'DJA2\blog'. По домашнему пути
откроется главная страница сайта, на которой отображенана основная информация,
реализованы "карусель", "шторка", "карточки", переходы на другие страницы
("Контакты", "Альбом V").
